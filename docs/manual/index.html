QQ adpter component plugin script software Mail

adapter

适配器,适配器可以令MiraiTravel拥有更多的能力。

component

组件,适配器开放接口,有统一的规则。

plugin

插件,根据组件提供的标准进行开发。拥有迁移的能力。

script

脚本,面向控制对象进行开发,开发时身临其境。如同在创造位于元宇宙的另一个自己。

software

软件,可以令 MiraiTravel 拥有更多的操作空间,简化 MiraiTravel 的部分操作。
